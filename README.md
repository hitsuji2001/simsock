## Description

Simple socket io implementation in C. **This only work on Linux**

## Quick start

```bash
$ ./build.sh --server
$ ./build.sh --client
```
Run these 2 commands on 2 differents terminals

```bash
$ ./server
```

```bash
$ ./client
```
